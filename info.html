Poo Poo

Tells the web browser to expect an HTML document. The root of the HTML document and parent of all other HTML elements on the webpage. Enclose other metadata about the site, such as its title. Contains the site's title, which is one way users can find your site through a search engine, like Google. Tells the web browser which character set to use. In this case, the character set is "utf-8".
